uOttaFit



project :

react fitness app

started :

7 february 2020

completed :

9 february 2020

a reactJS application which helps beginner and advanced athletes avoid weightlifting injuries by using the nuiTrack API alongwith a 3-dimensional camera to help weightlifters have the correct form. An artificial intelligence notifies the user when they should correct their form through a voice notification and instructs the user what changes they need to make. For example, when performing a deadlift, the app will notify the user that they should bend their knees mroe and straighten their back if their back is bent. I had a lot of fun making this project with my friends and I made it over the span of 2 days at my first hackathon, uOttaHack, hence the name of the project, uOttaFit.
